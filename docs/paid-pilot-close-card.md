# Paid Pilot Close Card (Day 9)

Use this card at the end of demo/discovery calls.

## Close Ask (Primary)
"We recommend a 14-day paid pilot for one workflow lane with weekly KPI reviews. If we hit the locked success criteria, we expand. Are you comfortable moving to pilot kickoff this week?"

## If They Ask About Price
"Pilot fee is typically $2,500 for one lane, with scope-adjusted options between $2,000 and $5,000 depending on support depth."

## If They Ask About Risk
"The pilot is narrow by design: one lane, one owner, KPI-gated success, weekly checkpoints, and no broad workflow replacement."

## If They Ask What They Need To Commit
1. Pilot owner
2. Week-0 baseline metrics
3. One weekly review slot
4. Signature on pilot terms

## Objection Fast Responses
- "We already have software" -> "We optimize pre-submit readiness and routing before your existing workflow."
- "Too much change" -> "One lane + one user to start."
- "How do we prove value" -> "Weekly KPI snapshots and before/after worksheet."

## Close Confirmation Checklist
- [ ] Verbal yes to paid pilot
- [ ] Terms sheet shared
- [ ] Kickoff date proposed
- [ ] Decision owner confirmed
- [ ] Next-step date set
