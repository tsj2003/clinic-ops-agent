# Case Study Draft v1 — AuthPilot AI (Pilot)

## Headline
AuthPilot AI reduced manual prior-auth prep friction in a targeted specialty workflow lane by converting payer policy/routing research into a structured operator handoff.

## Customer Profile
- Segment: Specialty clinic (spine/pain)
- Workflow lane: High-friction imaging prior auth
- Users: Prior-auth/RCM staff

## Problem
Before AuthPilot, staff manually:
1. Read payer policy pages
2. Mapped chart evidence to payer requirements
3. Looked up the correct payer routing/contact path

This caused variability, avoidable rework, and delayed submissions.

## Implementation
- Scope: 14-day paid pilot, one workflow lane
- Product setup: Guided intake + live TinyFish policy and contact workflows
- Outputs used by staff:
  - Submission readiness verdict
  - Missing evidence list
  - Operator handoff packet
  - KPI dashboard snapshot
